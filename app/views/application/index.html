Color

You can select the color you want for the background and text. The background color always will be the first paramater and is optional (default is #CCCCCC), the second parameter will be the text color, also optional (default will be B/W depending on te brightness of the background color). Example:
http://gummyimage.com/300x200/228C48/186333
We have 4 formats, the regular one that will be simple HEX format: 228C48. The 1 character format: C, this will be expanded to CCCCCC. The 2 character format: FA, this one will be translated to FAFAFA. And the 3 character format: AC5, this one will be translated to AACC55
http://gummyimage.com/600x400/AB/228C48
http://gummyimage.com/600x400/C4/3A6
http://gummyimage.com/600x400/A/5
