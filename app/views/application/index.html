Size

The size of the image is the first argument. This is in W x H for example: 800x600
Gummy image also supports aspects in X:Y format in Width or height for example 1920x10:9

Color

You can select the color you want for the background and text. The background color always will be the first paramater and is optional (default is #CCCCCC), the second parameter will be the text color, also optional (default will be B/W depending on te brightness of the background color). Example:
http://gummyimage.com/300x200/228C48/186333
We have 4 formats, the regular one that will be simple HEX format: 228C48. The 1 character format: C, this will be expanded to CCCCCC. The 2 character format: FA, this one will be translated to FAFAFA. And the 3 character format: AC5, this one will be translated to AACC55
http://gummyimage.com/600x400/AB/228C48
http://gummyimage.com/600x400/C4/3A6
http://gummyimage.com/600x400/A/5

Text

You can set the text whatever you want by passing as argument: text=myText. By default the text will be the size of the image. Keep in mind that some characters need to be encoded as URL, for example: spaces to %20 or using the simbol +
http://gummyimage.com/300x200/008CD7?text=gummy+image
As you see you can set a lot of different characters but this characters should be supported by the font. in this case is Droid sans
http://gummyimage.com/600x400/C4/3A6?text=(-_-/!Gopher!\-_-)
http://gummyimage.com/600x400/C4/3A6?text=Check%20this%20out
http://gummyimage.com/600x400/A/5?text=·wóòôów'

Format

There are 3 formats available for the images, these are PNG, GIF and JPG. By default gummy image uses PNG format. You can set the format with dot notation (.png, .gif, .jpg, .jpeg, .PNG, .GIF, .JPG, .JPEG) in every place you want, this means for example at the end of the url (before text param: AFAFAF.png) or in the middle of any color, yup thats od but you can: AFA.pngFAF
